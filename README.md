# Text Analysis Tool - CS1103 Assignment

## 📘 Overview
This repository contains the **Text Analysis Tool** developed by **francisMTech-ux** as part of the CS1103 Practical Assignment.  
The project demonstrates fundamental skills in Python programming, focusing on string handling, text analysis, and user interaction.

---

## ✨ Features
- **Character Count**: Calculates the total number of characters in the input text.
- **Word Count**: Calculates the total number of words.
- **Most Common Character**: Identifies the most frequently occurring character.
- **Character Frequency**: Allows the user to check how many times a specific character appears (case-insensitive).
- **Word Frequency**: Allows the user to check how many times a specific word appears (case-insensitive).
- **Unique Words**: Displays the number of unique words in the text.

---

## 🛠️ Requirements
- Python 3.8 or higher  
- No external libraries required (uses Python’s built-in functions and `collections.Counter`)
